# Sponsor NyaTerm

Thank you for supporting NyaTerm. Sponsorship helps with ongoing feature development, compatibility testing, documentation, release packaging, and infrastructure costs.

Sponsorship is completely optional and does not affect open-source access to NyaTerm. Feedback, bug reports, documentation improvements, code contributions, and sharing the project are also meaningful ways to help.

- Star the project on GitHub
- Report bugs or suggest improvements
- Improve documentation, translations, or usage examples
- Share NyaTerm with people looking for a remote terminal workspace
