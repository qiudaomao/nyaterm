# 赞助 NyaTerm

感谢你愿意支持 NyaTerm。这个项目会持续投入到功能开发、兼容性测试、文档维护、打包发布和基础设施成本上。你的赞助会帮助项目更稳定地迭代，也让维护者可以把更多时间放在实际体验改进上。

赞助完全自愿，不影响 NyaTerm 的开源使用。无论是提交反馈、参与讨论、贡献代码、完善文档，还是把项目分享给需要的人，都是对项目很重要的支持。
